Lesson 4 (Свойства float, overflow, cемантика)

SEO (Search engine optimization) - комплекс мероприятий направленных на поднятие позиции сайта в результатах выдачи поисковых систем.

SEO-оптимизация это отдельная отрасль со своими специалистами. Большинство мероприятий направленных на поднятие позиции сайта осуществляется после запуска сайта и связано с дополнительным вложением средств. Мы этим не занимаемся.

Наша задача будет сводиться лишь к тому чтобы придерживаться базовых принципов семантической верстки при порезке страницы:

  1. Применять HTML-теги по прямому назначению;
  2. Расставлять заголовки в правильном порядке;

HTML5 - это новая версия языка HTML, которая нам принесла:

  • новый doctype;
  • новые элементы html;
  • новые атрибуты;
  • поддержку аудио и видео;
  • полную поддержку CSS3;
  • 2D\3D графику;

Одна из основных задач HTML5 - добавление логики общей разметке HTML-страниц с целью упрощения программного анализа сайтов. В предыдущих стандартах для общей разметки страницы использовались теги <div>, которые не несут логической нагрузки, поэтому не было никакой возможности сделать блок header отличимым от блока content для программного анализа. В HTML5 для подобных целей используются специальные теги, которые в отличии от <div> несут и логическую нагрузку. Например теги: <header>, <footer>, <aside> и т.д.

тегатрибутытипописание
main grouping основной контент на странице; замена <div id="main"> или <div id="content">; один на страницу
section   section тематически сгруппированный контент, обычно с заголовком; логическая часть документа или статьи
article   section самостоятельная независимая часть документа, которая может легко распространяться (участвовать в рассылке) или быть повторно использованной на другом ресурсе (перепощена). Статья должна иметь смысл сама по себе и должна иметь возможность распространять ее независимо от остальной части сайта. Этот элемент задает содержание сайта вроде новости, статьи, записи блога, форума или др.
aside   section раздел страницы с дополнительным контентом, который косвенно относится к основному содержимому статьи или документа (зависит от того, во что вложен aside); сайдбар всего документа или дополнительная информация в статье\секции
header   section группа вступительных и навигационных элементов; обычно содержит заголовок; может содержать также логотипы, форму поиска, список содержимого; header документа\статьи\секции
footer   section колонтитул для ближайшего родителя; обычно содержит мета информацию, информацию об авторе, ссылки на соотносящиеся документы, копирайт; footer документа\статьи\секции
nav   section секция с навигационными ссылками; может иметь свой заголовок
figure   grouping самодостаточный контент, возможно с подписью, который ссылается на основное содержимое но при этом может быть легко перемещен по странице без потери смысла.
figcaption   grouping подпись к элементу figure
audio   embedded элемент для интеграции аудио
video   embedded элемент для интеграции видео
source   embedded источник видео или аудио, используется внутри соответствующих тегов.
time datetime="2009-11-13" text выделяет дату или время на странице, атрибут datetime должен содержать дату в правильном машинном формате
mark   text текст, выделенный цветом или любым другим способом
input type="email" forms выглядит как текстовое поле; валидирует введенные email адреса
input type="tel" forms выглядит как текстовое поле; валидирует введенные телефонные номера; переключает набор символов на мобильной клавиатуре
input type="url" forms выглядит как текстовое поле; валидирует введенные web адреса; подставляет http://
input type="search" forms выглядит как текстовое поле; позволяет очистить поле по клику на крестик
input placeholder="" forms определяет значение по умолчанию для текстовых полей формы;
