Lesson 14 (Git)

Git

  1. Устанавливаем git Скачать git

  2. Регистрируемся на github

  3. Создаем новый репозиторий

  1. git init

  2. Добавляем файл, проверяем состояние файлов

    git status

  3. Говорим git следи за этим файлом

    git add index.html

  4. Делаем слепок наших файлов

    git commit -m "first commit"

  5. Связываем наш локальный репозиторий с github

    git remote add origin https://github.com/AleksandrDyatlov/Test1.git

  6. Заливаем репозиторий на github

    git branch -M main

    git push -u origin main

  7. Стягиваем изменения с github в свой локальный репозиторий

  8. Изменяем наш файл

  9. Смотрим что в second commit

  10. Ссылка которую прикрепляем к ДЗ

  12. Удалить репозиторий

  13. git clone

  14. first init
